# 🌦️ ClimaSync - Modern Weather Dashboard

A sleek, responsive, and feature-rich Weather Application built with **Python (Flask)** and the **OpenWeatherMap API**. This project features a modern "Glassmorphism" UI and provides real-time meteorological data for cities worldwide.

---

## 🚀 Features

- **Real-Time Data:** Fetches live weather updates including temperature and weather conditions.
- **Advanced Metrics:** Displays "Feels Like" temperature, Humidity, Wind Speed, and Visibility.
- **Solar Data:** Provides accurate Sunrise and Sunset timings for the searched location.
- **Dynamic Icons:** Automatically updates weather icons based on current atmospheric conditions.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
- **Secure Integration:** Uses environment variables to protect sensitive API credentials.

---

## 🛠️ Tech Stack

- **Backend:** Python 3.10+, Flask
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), Jinja2 Templating
- **API:** OpenWeatherMap API
- **Icons:** FontAwesome 6.4 & OpenWeather Icon Set
